# gx430tctl

`gx430tctl` is the command dispatcher for GX430T Mac Control.

## Commands

```bash
bin/gx430tctl status
bin/gx430tctl discover
bin/gx430tctl install-local
bin/gx430tctl install-client 192.168.0.158
bin/gx430tctl share-on
bin/gx430tctl print-code128 1234567890 1
bin/gx430tctl print-code39 1234567890 1
bin/gx430tctl print-qr 1234567890 1
bin/gx430tctl diagnose
bin/gx430tctl verify
````
