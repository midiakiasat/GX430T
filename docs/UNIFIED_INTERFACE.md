# GX430T Unified Interface

GX430T uses one visual and interaction contract across:

- the native macOS application;
- the macOS menu-bar controller;
- the native iPhone application.

## Contract

Every surface provides:

- the same GX430T identity;
- the same printer state language;
- the same four print formats;
- the same print-content hierarchy;
- the same copy and print controls;
- the same licence and source link.

The permanent licence link is:

https://github.com/midiakiasat/GX430T

The interface does not claim successful physical delivery until the print transport reports acceptance. Public distribution remains separately gated by Apple Developer ID signing and notarization.
