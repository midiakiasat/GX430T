## Workplace use

1. Open the PDF presentation.
2. Explain the Mac host, colleague Mac, and iPhone surfaces.
3. Share the colleague quick-start document.
4. Keep the physical printer and print host inside the approved workplace network.
5. Use only an approved signed and notarized macOS package for production installation.

## Distribution boundary

This public release contains presentation and onboarding material only.

It does not publish:

- a production-authorized macOS installer;
- a globally distributable iPhone IPA;
- workplace printer credentials;
- pairing secrets;
- internal host addresses;
- unrestricted access to the physical GX430t printer.

The macOS distribution candidate remains an unpublished draft until Developer ID signing, notarization, clean-machine installation, pairing, and physical print acceptance are complete.
