# GX430T — Workplace Presentation

## One sentence

GX430T turns one USB-connected Zebra GX430t into a controlled Mac and iPhone
label-printing service for approved workplace users.

## The problem

The Zebra GX430t is reliable, but normal operation requires knowledge of USB
printer configuration, CUPS, IPP, ZPL, printer queues, network sharing, and
scanner-safe barcode settings.

That creates setup risk, inconsistent labels, support overhead, and dependence
on one technically experienced operator.

## The solution

GX430T provides:

- a native Mac application;
- a Mac menu-bar print host;
- an iPhone application;
- guided host and colleague installation;
- secure device pairing;
- scanner-safe barcode templates;
- controlled copy counts and previews;
- printer discovery and repair;
- diagnostics and evidence export;
- signed and versioned distribution;
- one official download location.

## How it works

1. One host Mac connects to the Zebra GX430t by USB.
2. GX430T verifies and manages the printer queue.
3. GX430T Print Host shares approved printing on the workplace network.
4. Colleague Macs and paired iPhones submit controlled label jobs.
5. The host Mac sends verified ZPL to the physical printer.

## What colleagues do

1. Download GX430T from the official release link.
2. Install the Mac or iPhone app.
3. Enter the host address.
4. Pair using the six-digit code.
5. Preview and print.

## Safety boundary

The physical printer remains private to the workplace network.

GX430T does not expose the printer directly to the public internet and does not
turn the printer into a cloud-managed device.
