
# Pro Operator Workflow

1. Open GX430T Mac Control.
2. Drop/upload Excel or CSV.
3. Check queue count and preview.
4. Press **Print next** for controlled printing or **Print all queued** for batch mode.
5. Printed jobs move out of the active queue.
6. Failed jobs stay visible with error status.

The system is local-first and does not send uploaded files outside the Mac.
