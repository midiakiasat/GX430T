# GX430T Colleague Quick Start

## What you need

- A supported Mac.
- The GX430T Mac installer.
- The GX430T iPhone app when mobile control is required.
- The host Mac address supplied by the GX430T administrator.
- A six-digit pairing code generated by the host Mac.
- Access to the same approved workplace network.

## Install on Mac

1. Open the official GX430T release page.
2. Download the current signed Mac installer.
3. Verify the checksum when supplied.
4. Open the installer and complete the macOS prompts.
5. Launch **GX430T** from Applications.
6. Choose **Colleague Mac**.
7. Enter the host Mac address.
8. Confirm the shared printer is detected.
9. Run **Connection Test**.
10. Print one test label.

## Install on iPhone

1. Use the official installation link supplied by the GX430T administrator.
2. Install **GX430T**.
3. Open the app.
4. Enter the host Mac address.
5. Enter the six-digit pairing code shown on the host Mac.
6. Confirm the device name.
7. Complete pairing.
8. Send one test label.

## Print

1. Select the paired Mac host.
2. Choose the label type.
3. Enter or scan the label value.
4. Check the preview.
5. Select the number of copies.
6. Press **Print**.
7. Confirm the label printed correctly.

## The host Mac must remain

- powered on;
- connected to the approved network;
- connected to the Zebra GX430t by USB;
- running GX430T Print Host;
- permitted to share the printer.

## Support information to send

When support is required, send:

- Mac model and macOS version;
- GX430T app version;
- iPhone model and iOS version, when applicable;
- host Mac address;
- exact error text;
- GX430T diagnostics export;
- whether a local Mac test print succeeds.
